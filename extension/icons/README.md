# Icons

Thay 3 file PNG thật vào đây trước khi load extension:
- `icon16.png` (16×16)
- `icon48.png` (48×48)
- `icon128.png` (128×128)

Tạm thời có thể dùng bất kỳ ảnh vuông nào đổi tên cho đủ 3 kích thước, hoặc
xoá phần `"icons"` và `"default_icon"` trong `manifest.json` nếu muốn load nhanh để test.
